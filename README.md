# Pixar Animation App

Pixar Animation App adalah sebuah aplikasi yang menyediakan informasi lengkap tentang film-film dari studio animasi terkenal, Pixar Animation Studios. 
Aplikasi ini memudahkan pengguna untuk menelusuri dan mengetahui lebih banyak tentang film-film yang telah diproduksi oleh Pixar.
